# MBO 목표서 — Sensor Gateway 고도화
승인: 2026-06-18

## 최상위 목표
"Sensor Gateway를 현장 엔지니어가 즉시 현황 판단할 수 있는 수준으로 고도화 — 태그 실시간 트렌드·알람·장치 진단·프로토콜 파라미터를 추가한다"

- Node-RED 탭 개선 (msg/s 바차트 + 에러 카운트)
- 빌드 + 배포
